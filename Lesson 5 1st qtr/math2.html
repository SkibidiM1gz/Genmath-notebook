(-3, 3)

f(x) < 0:

  • Give the corresponding x-values whose y-values are negative
  • Look at the graph below the x-axis

For the graph on the right, the values of x where f(x) is less than zero is the interval…

(-∞, -3) ∪ (3, ∞)

or

{ x | x ∈ ℝ, x < -3 ∪ x > 3 }

INTERCEPTS

X-INTERCEPT

  • An x-intercept is the x-coordinate of the point where the y-value is zero, which means it is the x-coordinate of the point where the graph crosses the x-axis.

NOTE: A function can have multiple x-intercepts.

Y-INTERCEPT

  • A y-intercept is the y-coordinate of the point where the x-value is zero, which means it is the y-coordinate of the point where the graph crosses the y-axis.

NOTE: A function can only have one y-intercept. If it has two or more y-intercepts, then it is not a function because it will fail the vertical line test.

ZERO OF A FUNCTION

  • The "zero" of a function is the value k, wherein f(k) = 0 (an input value that produces an output of zero).

NOTE: x-intercept is the same as the ZERO of a FUNCTION.

EXAMPLE

Given the graph of f on the right, determine the following characteristics of f.

  1. Domain of f
  2. Range of f
  3. When is f(x) ≥ 0?
  4. On what interval(s) is f increasing?
  5. On what interval(s) is f decreasing?
  6. When is f(x) < 0?
  7. What is/are the x-intercept/s?
  8. What is the y-intercept?

EXAMPLE (solution)

Domain of f

Answer:

( -∞ , 3 ]

or

{ x | x ∈ ℝ, x ≤ 3 }

EXAMPLE (solution)

Range of f

Answer:

( -∞ , 4 ]

or

{ f(x) | f(x) ∈ ℝ, f(x) ≤ 4 }

or

{ y | y ∈ ℝ, y ≤ 4 }
